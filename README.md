# Sistema de gestion para la papeleria mile

Aplicación web desarrollada como proyecto académico, enfocada en la gestión de proveedores , usuarios, ventas, clientes , empleados  y inventario.

## Funcionalidades
- Registro de productos
- Gestión de inventario
- Gestión de prveedores
- Gestión de usuarios
- Gestión de clientes
- Gestión de empleados
- Gestión de ventas
- CRUD completo
- Interfaz responsive

## Tecnologías utilizadas
- PHP
- MySQL
- HTML
- CSS
-JavaScript

## Participación
Desarrollo completo del frontend y backend del sistema.
